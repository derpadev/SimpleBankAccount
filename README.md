# SimpleBankAccount

A simple Java banking system that supports Standard and VIP accounts.

## Features
- Create Standard or VIP accounts
- Deposit and withdraw money
- Display account details
- Calculate simple and compound interest
- Remove accounts

## Project Structure

```
SimpleBankAccount/
├─ src/       # Java source code
├─ javadoc/   # Generated JavaDoc HTML
├─ README.md  # This file
```

## How to Run
1. Compile the source code in `src/`
2. Run the `BankSystem` class
3. Follow the menu to interact with accounts

## JavaDoc
The JavaDoc is included in the `javadoc/` folder. Open `index.html` in a browser to view.
